当前进行较长时间操作如install时，界面仅在tab行下显示了一串字符信息，不显眼，也会造成用户可以反复快速进行操作大量spawn后台cli的问题。请在进行耗时操作时：

1. cli应反馈progress，包括给人类看的和json结构化的

2. UI应在耗时操作时显示一个弹出面板，其中显示cli返回的progress信息。该面板应该为模态，暂时屏蔽用户的进一步操作。

3. 但模态面板应考虑到cli因网络等问题卡死时的超时措施。默认超时30s，超时后提供给用户按q或esc关闭面板的选择，也可继续等待。

4. 最终结果不应在list上方简单显示一行文字，不太显眼。应该在progress 面板中显示结果。
